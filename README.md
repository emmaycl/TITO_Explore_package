# TITO_Explore
Python tools for working with translation-invariant total orders (TITOs).

## Functions include: 

- normalization: given any window notation, normalize to the standard form that we defined in the paper. 
- comparison: compare the extended weak order of two given TITO objects. 
- inversion sets: find the inversion sets of two given TITO objects. 
- join computation: compute the least upper bound of two given TITO objects.
